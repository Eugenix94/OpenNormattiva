# 🔎 OpenNormattiva — Search

A production search UI for the Italian legislative corpus (Normattiva).

**Purpose:** Fast public search and retrieval for Italian laws with full-text search, filters, and citation browsing.

**Dataset:** diatribe00/normattiva-data
